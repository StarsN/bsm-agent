# 常见陷阱模式

在开仓前必须对照以下模式，若高度共振，必须执行相应策略。

## 陷阱一：流动性吸收 / 强 Taker 诱多
**系统标签**：`buy_pressure_faded`（买盘消退）

**情境**：Taker（主动买方）力量（如主动买卖比、买入占比）呈现极端强势，但短期多周期价格（如 1h、24h）表现停滞甚至微跌，盘口呈现显著的“量价背离”。
**倾向**：极大概率是散户的市价买盘被暗处的限价单（Maker 冰山委托）持续吸收，主力资金正在隐蔽派发或制造诱多假象。这是实盘中最经典的流动性陷阱，此时追多胜率极低。
**策略**：面对此种结构性背离，**必须暂时搁置多单计划**。绝不可仅凭 Taker 强势就左侧入场，必须等待价格实体有效突破阻力区，并与 Taker 动能形成同向共振后，再重新评估。

---

## 陷阱二：趋势反噬 / 短周期 OI 暴增 + 价格暴跌
**系统标签**：`oi15_reversed` / `oi1h_reversed` / `oi4h_reversed`

**情境**：短周期内（如 15m 或 1h）未平仓合约（OI）出现异常的大幅飙升，但同期价格未见上涨，反而显著下跌或在高位遇阻快速回落。
**倾向**：短期内有大量新空头资金强势入场，或主力多头在派发过程中开空进行套期保值。这种“OI 增量与价格下跌的共振”说明空头掌握了绝对主动权，是强烈的阶段性顶部信号。
**策略**：**严格规避做多操作**。切勿左侧接飞刀，需耐心等待 OI 潮水退去（爆仓去杠杆完成）或价格构筑有效底部、确立新方向后再做决策。

---

## 陷阱三：杠杆清算 / 极端费率叠加极端位置

**情境**：资金费率呈现极端正值（多头极度拥挤）或极端负值（空头极度拥挤），且价格已经历了单边大幅度拉升或暴跌，当前处于结构性的高位或低位区间。
**倾向**：衍生品市场情绪极度过热，持仓高度失衡。
	极端正费率 + 高位 = 随时可能引发多头踩踏（Long Squeeze）清算，导致瀑布式下跌。
    极端负费率 + 低位 = 酝酿着暴力的向上轧空（Short Squeeze），但也可能是空头趋势的终极加速期。
**策略**：
    当**正费率极端且价格处于高位**时，绝对禁止开启新多单。
    当**负费率极端且价格处于低位**时，若出现 OI 稳步上升，可寻找右侧多单机会，但必须有现货盘口或其他维度信号的强力确认；若无明确共振信号，坚决观望。

---

## 陷阱四：滑点黑洞 / 盘口流动性匮乏

**情境**：当前盘口深度（买一/卖一或 1% 内挂单量）相对于系统拟开仓的资金体量过于单薄，且买卖价差（Spread）显著宽于大盘平均水平。
**倾向**：该资产流动性极差，微小的市价单即可能引发剧烈波动。入场极易遭遇巨大的滑点损耗，且在触发止损时大概率找不到足够的对手盘，导致实际亏损被动放大，严重破坏系统的盈亏比模型。
**策略**：**必须动态评估流动性成本**。应根据当前的盘口深度强制降级持仓档位（如降至 `quarter` 甚至更低）。若评估滑点成本过高，应直接放弃交易，坚决规避流动性击穿风险。
