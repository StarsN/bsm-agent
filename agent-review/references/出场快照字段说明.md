# 出场快照（dimension_data）字段说明

平仓 journal 的 `dimension_data` 与开仓格式一致——中文平铺 JSON，多一个 `平仓价` 字段。

格式：
```json
{
  "平仓价": 0.00123,
  "标记价": 0.00120,
  "15分钟涨跌": 1.5,
  "1小时涨跌": 3.2,
  ...
}
```

## 全部字段

| 字段 | 含义 |
|------|------|
| 平仓价 | 平仓成交价，可能含假设滑点 |
| 标记价 | 平仓时合约标记价 |
| 15分钟涨跌 | 15分钟价格变化(%) |
| 1小时涨跌 | 1小时价格变化(%) |
| 4小时涨跌 | 4小时价格变化(%) |
| 24小时涨跌 | 24小时价格变化(%) |
| 48小时涨跌 | 48小时价格变化(%) |
| 资金费率(%/8h) | 资金费率 |
| 未平仓(USD) | 未平仓合约金额 |
| OI15分钟变化 | OI 15分钟变化(%) |
| OI1小时变化 | OI 1小时变化(%) |
| OI4小时变化 | OI 4小时变化(%) |
| OI48小时变化 | OI 48小时变化(%) |
| 主动买卖比 | taker buy/sell 总量比 |
| 主动买入占比 | taker buy / total (%) |
| Taker趋势 | taker 最近 vs 较早趋势(%)，正=买盘增强 |
| 盘口价差 | 买卖价差(%) |
| 买盘深度(USD) | 1%内买盘深度 |
| 卖盘深度(USD) | 1%内卖盘深度 |
| 盘口失衡度 | 深度偏向(%)，正=买盘多 |
| 24小时成交额 | 24小时成交量(USD) |
| 散户多空比 | 全网多空账户比 |
| 大户多空比 | 大户持仓多空比 |
| 信号标签 | 分析标签列表 |
| 分析备注 | 分析解读列表 |
| OI背离 | OI 48h 背离检测。`null` 表示无背离。命中时结构见下方 |

**OI背离 命中结构**（条件：\|48h OI\| ≥ 20% 且 \|48h 价格\| < 3%）：

```json
{
  "type": "oi_accumulation" / "oi_distribution",
  "direction": "积累中（多头/空头未定）" / "撤离中",
  "oi_pct": 25.3,
  "price_pct": 1.22,
  "note": "48h 内 OI 增加 +25.3% 但价格仅 +1.2%——资金在进场但行情未启动"
}
```

## 入场快照在哪

入场快照存在同一条 **开仓 journal 的 `dimension_data`** 中（通过 `order_id` 配对），格式相同，无 `平仓价` 字段。
