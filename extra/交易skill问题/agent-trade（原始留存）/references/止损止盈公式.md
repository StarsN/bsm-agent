# 止损止盈设定

## 止损

- 距离范围：1.5% - 5%（参考 ATR）
- < 1%：太紧，正常波动扫掉
- > 5%：太宽，单笔风险过大
- SL 必须 < entry，否则系统拒绝

系统（auto_trader）会校验你给的 SL/TP。不合理时自动用 ATR 兜底。

## 止盈

- TP1 = entry + (entry - SL) × 1.5    （+1.5R）
- TP2 = entry + (entry - SL) × 3.0    （+3R）
- 保证盈亏比 ≥ 2:1

系统执行时：TP1 触发平 30%、TP2 触发再平 30%、剩余 40% 跟踪止盈。

## 仓位

你只决定 tier（full / half / quarter），具体数量由 risk.py 按风险反推。
- full：风险 = equity × 1%
- half：风险 = equity × 0.5%
- quarter：风险 = equity × 0.25%
