# 常见陷阱模式

这些是在实盘中反复出现、导致亏损的模式。开仓前必须对照检查。

## 陷阱一：强Taker买盘 + 价格不动

**表象**：Taker主动买卖比很高（>1.5），主动买入占比高（>60%），但价格横盘或微跌。

**本质**：大资金在对倒制造虚假买盘，或在低位缓慢吸筹但尚未到拉升阶段。

**系统标签**：`buy_pressure_faded`（买盘消退）

**判断标准**：
- Taker > 1.5 且 pct > 60%
- 同时 1h 价格涨跌 < 1% 或为负
- 24h 价格涨跌 < 5%

**处理**：**绝对不开仓**。即使其他信号良好也不开。这个模式在实盘中反复杀人。

**实例**（2026-05-06）：
- FET：Taker 1.98 / 买盘66.4%，4h仅+1.7% → 当日止损
- HYPE：Taker 2.28 / 买盘69.5%，价格不跟 → 当日止损
- WLD：Taker 1.75 / 买盘63.6%，24h仅+3.7% → 避开了（tag_stats 中 buy_pressure_faded 活跃）

## 陷阱二：短周期OI暴增 + 价格下跌

**表象**：15m或1h OI大幅上升（>3%），同期价格下跌。

**本质**：新空头大量入场，或主力在派发中开空对冲。量价背离是顶部信号。

**系统标签**：`oi15_reversed` / `oi1h_reversed` / `oi4h_reversed`

**判断标准**：
- 某周期 OI 变化 > +2% 且同期价格变化 < 0% 或为负
- 多周期背离比单周期更危险

**处理**：不开多单。等OI回落或价格确认方向。

## 陷阱三：高费率 + 价格高位

**表象**：资金费率极高正值（>0.1%）或极高负值（<-0.5%），价格已大幅上涨。

**本职**：极高正费率 = 多头拥挤，随时可能踩踏。极高负费率 = 空头拥挤，可能有轧空但也可能空头是对的。

**判断标准**：
- 正费率 > 0.1%：多头拥挤，不开新多
- 负费率 < -0.5%：极端空头拥挤，需结合价格位置判断（高位 = 空头可能对，低位 = 可能轧空）

**处理**：正费率极高 + 价格高位 → 不开。负费率极端 + 价格低位 + OI上升 → 可考虑，但需其他信号确认。

## 陷阱四：薄盘口

**表象**：盘口深度很浅（bid或ask < $100k），价差大（>0.05%）。

**本质**：流动性差，入场容易滑点，出场可能找不到对手盘。

**判断标准**：
- depth_bid 或 depth_ask < $100,000
- spread > 0.05%

**处理**：不开仓或仅 quarter 档。薄盘口币止损容易被击穿。
