# 复盘报告模板

推送时用反引号代码块包裹，TG 会用等宽字体渲染，表格才能对齐。

```
📊 Agent 每日复盘 {UTC日期} (UTC+8 {北京日期})

━━━━━━━━━━━━━━━━━━━━━━━━
本次统计（仅 Agent 决策）
  开仓: {N} 笔  平仓: {N} 笔
  盈利: {N} 笔  亏损: {N} 笔
  本次 PnL: ${xxx} ({x.xx}%)
  胜率: {xx}%

━━━━━━━━━━━━━━━━━━━━━━━━
交易明细
  ✅ {TOKEN} +{x.xx}% — {一句话}
  ❌ {TOKEN} -{x.xx}% — {一句话}

━━━━━━━━━━━━━━━━━━━━━━━━
新教训
  1. [{severity}] {token}: {lesson} → rule: {rule_update}
  2. ...

━━━━━━━━━━━━━━━━━━━━━━━━
模式发现
  - {共性问题或好习惯}
  - {与已有教训的关联}

📊 本次无交易   （无操作时）
```
